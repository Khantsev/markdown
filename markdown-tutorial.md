# Markdown tutorial

---

### Header

```
# Head 1
## Head 2
### Head 3
#### Head 4
##### Head 5
##### Head 6
```
*Example:*

### Head 3
#### Head 4
---

### Highlight

```
_Italic_ and *italic*

__Bold__ and **bold**

~~Line throw~~
```

*Example:*

_Bird_ and *bird*

__Elephant__ and **elephant**

~~Error~~

---

### List

```
* Item 1
* Item 2
  * Item 2.1
  * Item 2.2
* Item 3

1. Item 
2. Item
3. Item
```
*Example:*

Bulleted list

* Section 1
* Section 2
  * Subsection 2.1
  * Subsection 2.2
* Section 3

Numbered list

1. MacBook Air 
2. iPad
3. iPhone

---

### Link

```

```
*Example:*

[Google](https://google.com)
